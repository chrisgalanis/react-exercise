## Exercise 

Φτιάξε μια NextJS εφαρμογή (ακολουθώντας τα official NextJS docs) που να χρησιμοποιεί axios (ένα node package) για να κάνει API request στο https://be.givelink.app/data και να προβάλει την πληροφορία προϊόντων σε μια σελίδα. Η σελίδα να έχει μορφή A x B grid, με κουτάκια που δείχνουν το κάθε προιον. Όταν ο χρήστης κάνει hover ενα προιον, αυτό θα πρέπει να αλλάζει (ελαφρώς) χρωμα για να δεινει feedback στον χρήστη οτι ειναι clickable. Υπάρχουν 300+ προϊόντα στη λίστα, αλλά θα πρέπει να προβάλλονται μόνο τα ενεργά: product.active = true. Έχεις 1 εβδομάδα να μας στείλεις τα παραδοτέα.
